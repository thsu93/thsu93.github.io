ABOUT ME



I'm a current medical student who's looking to transition from medicine into work in software engineering.

I've been coding since I was in middle school, and have a history of writing code for genomic data analysis and neuroradiology imaging processing.

In my free time, I enjoy building interesting programs and proof of concept video game prototypes in Unity, Unreal Engine, Godot, and RenPy.

Take a look at some of them down below!


RECENT PROJECTS

Fighting Game X Platformer Prototype

Framework and engine for a 2-D Character Action game.
Designed as a single-player RPG with platforming movement and fighting game mechanics, allowing for more of a free-flowing and heavily customizable yet still relatively forgiving combat system than could be present in a traditional player-vs-player fighting game or a standard platformer.

Coded using GDScript, created in the Godot game engine

Noms Calorie Diary
filler here

Calorie Calculator and Diet Diary using the NOMS library to draw dietary information from the USDA's FoodData Central API, collate it for nutritional analysis and compare it to Recommended Daily Intake values, and export it to JSON files.

Work In Progress:
Currently updating to change from a CLI to GUI using Python's built-in TKINTER library

Coded using Python

Simultaneous-Play Card Game Engine Prototype
filler here

Prototype of a single or two-player deckbuilding card game. Mechanics focused on selecting, hiding, and then simultaneously revealing different cards in play. Tracks mouse movements, playzones, and card history. Allows for persistent field effects and card upgrading.

Coded using C# in the Unity game engine

Isometric Roguelike Project
